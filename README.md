# IBM Naan Mudhalvan – Day 4 Task
## Library Management System (Backend)

This project is developed as part of the IBM Naan Mudhalvan Internship – Day 4 Knowledge Transfer task.

### Project Description
A simple Library Management System backend built using:
- Node.js
- Express.js
- MongoDB (MongoDB Atlas)

The application supports CRUD operations on books using REST APIs.

### Features
- Add new books to the library
- View all books
- Filter books by category
- Get books published after 2015
- Update available copies of books
- Automatically delete books when copies reach zero
- Delete books based on conditions

### Tools & Technologies Used
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- Postman
- Replit

### API Testing
All APIs were tested using Postman and verified successfully.
